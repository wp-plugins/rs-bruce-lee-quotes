A plugin for displaying a random quote by Bruce Lee, on your site via a widget

== Description ==

Simply add the RS Bruce Lee Quotes Widget to your required place and a quote will appear at random each page load.

== Installation ==

1. Upload `bruce-lee-quotes` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
